Wasserdruck

  • Version der Software
    "/version.html"
  • Controller/Server Info
    "/info.html"
  • Erfrage Intervall der Messungen
    • "/api/v1/interval"
  • Daten von HEUTE
    "/api/v1/today"
  • Daten vom Datum YYYY-MM-DD (Year, Month, Day)
    "/api/v1/data?from=2024-07-06" Daten vom 06.07.2024 "/api/v1/data?from=2025-01-03" Daten vom 03.01.2025
  • Helligkeit des LED Stripe (0 - 255)
    "/api/v1/led"
  • Setzte Zeitzohne (hier Berlin, Normalzeit)
    "/api/v1/set-timezone?timezone-offset=3600"
  • Setzte Zeitzohne (hier Berlin, Sommerzeit)
    "/api/v1/set-timezone?timezone-offset=7200"
  • Setzte loglevel der App (via usb)
    "/api/v1/set-loglevel?level=7"

    EMERGENCY = 0
    ALERT = 1,
    CRITICAL = 2,
    ERROR = 3,
    WARNING = 4,
    NOTICE = 5,
    INFO = 6,
    DEBUG = 7
  • Setzte Messintervall in Sekunden
    "/api/v1/set-interval?interval=XX"
  • Setzte Helligkeit des LED Streifens (0 - 255)
    "/api/v1/set-led?brightness=XXX"
